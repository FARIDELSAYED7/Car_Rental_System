## شرح عربي مبسّط (للناس اللي أول مرة تتعلم برمجة)

الفكرة ببساطة: عندنا نظام لتأجير السيارات.

- **المستخدم** بيسجّل دخول (Admin أو Customer).
- **الـ Admin** يضيف/يعدل/يحذف العربيات.
- **الـ Customer** يشوف العربيات المتاحة، يحجز واحدة، ويدفع.
- بعد الحجز، النظام يعمل **فاتورة** فيها كل التفاصيل.

> **معلومة مهمّة عن AI:**
> المشروع الحالي لا يحتوي على ذكاء اصطناعي حقيقي. البحث والفلترة المنفّذة مجرد منطق برمجي بسيط (حلقات وشروط).
